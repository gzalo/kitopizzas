# Visor de modelos y niveles 3D

Los visores permiten cargar y visualizar los modelos MDL y niveles WDL usados en el juego.

Es necesario bajarse el repositorio, tener instalado Python 3 e instalar algunas bibliotecas. Para instalarlas, correr esto desde la carpeta `viewer/`:
```
pip install -r requirements.txt
```

Luego se pueden ejecutar con:
```
python mdl_viewer.py "C:\\Program Files (x86)\\KitoPizzas Vol 1\\"

python wmb_viewer.py "C:\\Program Files (x86)\\KitoPizzas Vol 1\\"
```
